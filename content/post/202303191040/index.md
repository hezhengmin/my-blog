1. 安裝 Node.js
2. 安裝 Vue CLI
```
npm install -g @vue/cli
```
3. 創建 Vue.js 專案
```
vue create my-project
```
4. 啟動 Vue.js 專案
```
cd my-project
npm run serve
```
